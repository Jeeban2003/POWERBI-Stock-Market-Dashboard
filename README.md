# Stock Market & Mutual Fund Power BI Dashboard

This project presents a financial analytics dashboard built using Power BI to analyze stock market and mutual fund data.

## Tools Used
- Power BI
- SQL (PostgreSQL)
- Microsoft Excel

## Dashboard Highlights
- Total AUM analysis
- 1M, 1Y, and 3Y return comparison
- Risk vs return analysis
- Asset class and fund category insights
- Fund size vs performance
- Minimum SIP vs return analysis

## Purpose
The dashboard is designed to demonstrate real-world financial data analysis and business intelligence reporting using Power BI.
